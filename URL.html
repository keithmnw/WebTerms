URL

A URL (Uniform Resource Locator) is the address of a web resource on the internet. It specifies the location of a resource and the protocol used to access it.

URLs are the fundamental addressing system of the World Wide Web, allowing users and browsers to locate specific resources such as web pages, images, videos, and other files.

URL Structure Breakdown

URL Components:

  • Scheme: The protocol prefix (http://, https://, ftp://, mailto:)
  • Domain: The server name (www.example.com)
  • Port: Optional port number (:8080)
  • Path: Location of resource on server (/folder/page.html)
  • Query String: Parameters passed to server (?id=123&lang=en)
  • Fragment: Anchor within the page (#section)

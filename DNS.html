Domain Name Server (DNS)

DNS stands for Domain Name System. It is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network.

DNS essentially acts as the phone book of the internet, translating domain names (like www.example.com) into IP addresses that computers can understand. This allows users to access websites using easy-to-remember names instead of numerical IP addresses.

DNS Query Resolution Process

How DNS works:

  • User Query: When you type a website address in your browser, your computer sends a DNS query to a recursive resolver.
  • Root Server: The resolver first contacts a root server, which directs it to the appropriate TLD server (like .com, .org).
  • TLD Server: The TLD server then directs the resolver to the authoritative name server for the specific domain.
  • Authoritative Response: The authoritative name server returns the IP address, which your browser uses to connect to the website.
